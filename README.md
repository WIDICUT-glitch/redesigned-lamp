# Social Marketplace Website

## Description
This is a simple web app that allows users to:
- Share posts (social interaction)
- Sell items (marketplace feature)

## Features
- Create and view posts
- Add items for sale
- Simple and clean interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open index.html in your browser
2. Write a post and click "Post"
3. Add an item and price, then click "Sell"

## Future Improvements
- User accounts (login/signup)
- Real-time chat
